# Adaptation mobile — Implementation Plan

> REQUIRED SUB-SKILL: superpowers:executing-plans.

**Goal:** Affichage en cartes des tableaux + confort tactile sur mobile, sans changer le desktop.

**Architecture:** Attributs `data-label`/`card-title`/`mobile-hide` dans les rendus (inertes sur desktop) + un bloc CSS `@media (max-width: 640px)`.

## Global Constraints
- **Zéro changement desktop** : tout le visuel mobile est dans `@media (max-width: 640px)`.
- Pas de build ; français.
- Tests headless : serveur `PYTHONIOENCODING=utf-8` ; localStorage seedé ; ouvrir `http://localhost:8750/terminal-tout-en-un.html`.
